# expertneuro-monitor

Автономный монитор CRM ExpertNeuro. **Чистые Python-скрипты, 0 токенов LLM** —
GitHub Actions по расписанию (5 раз в день) снимает снапшот Битрикс24 (только чтение),
считает метрики и обновляет дашборд на GitHub Pages.

## Доступ: сайт по ссылке + пароль

Дашборд публикуется на GitHub Pages (обычная ссылка), но **все данные зашифрованы
AES-256-GCM** (ключ выводится из пароля, PBKDF2-SHA256 · 300k итераций). При открытии
страница просит пароль; без него в репозитории и на сайте — только шифроблоб.
Пароль задаётся секретом `DASH_PASSWORD` и отправляется только своим.

В публичном репо НЕТ: вебхука Битрикса, имён сотрудников (карта менеджеров — в секрете
`BX_USERS_JSON`), открытых метрик (только `*.enc.*`).

## Что показывает дашборд (`docs/index.html`)

- **Алерты**: падение потока, «ничьи» лиды (без менеджера), менеджеры с провальной конверсией, обвал подрядчика день к дню.
- **Отчёт по каждому менеджеру**: сегодня и вчера — лидов / записал / конверсия / НБ / в работе / дожим / брак.
- **Подрядчики и города**: вчера → сегодня, с дельтами.
- **Тренд 8 дней**: лиды и записи.
- **Конвейер агентов**: статус последнего пакета гипотез (генератор → критик → контролёр → продакшн), публикуется `scripts/push_status.py`.

## Секреты (Settings → Secrets and variables → Actions)

| Секрет | Что |
|---|---|
| `BITRIX_WEBHOOK` | входящий вебхук Битрикс24 (scope crm, read-only использование) |
| `DASH_PASSWORD` | пароль дашборда (им шифруются данные) |
| `BX_USERS_JSON` | JSON-карта `{"id":"Имя"}` менеджеров |

## Устройство

| Файл | Роль |
|---|---|
| `scripts/monitor.py` | снапшот Битрикса → `docs/data/data.enc.js` (+ шифрованная история) |
|  `scripts/push_run.py` | журнал прогонов агентов → шифрованный runs.enc.js для страницы «Отдел» |
| `.github/workflows/monitor.yml` | cron 09/12/15/18/21 по Алматы + ручной запуск |

Методика: дедуп по контакту, «не засчитано» исключено из рекламных метрик (но входит в
нагрузку менеджеров), сегодняшний день «дозревает» (+13ч НБ→дожим). Правила — в рабочей
папке «Сверка» (`CLAUDE.md`, HANDOFF).

Локальный прогон без шифрования: `BITRIX_WEBHOOK=<url> python scripts/monitor.py`
(создаст открытый `docs/data/data.js`; в git не коммитится).
